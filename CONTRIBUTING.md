# Contributing

Contributions are welcome!

## Quick Guide

1. **Found a bug?** Open an Issue with reproduction steps.
2. **Want to add a feature?** Open an Issue to discuss it first.
3. **Ready to code?**
   - Fork the repo.
   - Create a branch (`git checkout -b feature/cool-thing`).
   - Run tests: `make test`.
   - Open a Pull Request.

## Style
- 4 space indentation.
- Standard C99.
- Keep it clean.
